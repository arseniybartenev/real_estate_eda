## Описание проекта

Данный проект посвящён исследовательскому анализу данных (EDA) архива объявлений о продаже квартир в Санкт‑Петербурге и соседних населённых пунктах, предоставленного сервисом Яндекс.Недвижимость.

Цель работы — изучить структуру данных, выявить закономерности и аномалии, а также подготовить признаки для последующего построения модели, определяющей рыночную стоимость объектов недвижимости. Результаты анализа помогут в разработке автоматизированной системы, способной отслеживать аномалии и потенциально мошенническую деятельность.

## Данные

В качестве исходных данных используется файл `real_estate_data.csv`. Он содержит информацию о квартирах, включая:

*   **Введённые пользователем:** общая площадь, жилая площадь, площадь кухни, количество комнат, этаж, цена и др.
*   **Автоматически полученные (картографические):** расстояние до центра, аэропорта, ближайшего парка и водоёма.

Всего в таблице 22 столбца и более 23 тысяч записей.

## Этапы работы

1.  **Первичный анализ и предобработка данных**
    *   Изучение общей информации, типов данных, наличия пропусков.
    *   Корректировка наименований столбцов.
    *   Обработка пропущенных значений (заполнение медианами, модами, обоснованная замена на 0).
    *   Приведение типов данных к корректным форматам (целые числа, дата, bool).
    *   Поиск и устранение явных и неявных дубликатов.

2.  **Выявление и обработка аномалий**
    *   Визуализация распределений числовых признаков с помощью гистограмм и ящиков с усами.
    *   Определение граничных значений для фильтрации выбросов (с контролем потерь данных < 1%).
    *   Коррекция аномальных значений (например, перенос запятой в высоте потолков).

3.  **Расчёт новых признаков (Feature Engineering)**
    *   Цена квадратного метра (`cost_per_sqm`).
    *   День недели, месяц, год публикации объявления (`weekday`, `month`, `year`).
    *   Категория этажа квартиры (`floor_category`: первый, последний, другой).
    *   Расстояние до центра в километрах (`city_centers_nearest_km`).

4.  **Исследовательский анализ данных**
    *   Анализ распределений основных параметров (площади, цена, количество комнат, высота потолков и др.).
    *   Изучение влияния различных факторов (тип этажа, удалённость от центра) на стоимость недвижимости.
    *   Выявление основных тенденций на рынке недвижимости.

## Основные выводы

*   **Типичная квартира** на рынке имеет общую площадь 50–60 кв.м, жилую – 30–35 кв.м, кухню – около 10 кв.м, 2 комнаты, стандартную высоту потолков (2,5–2,7 м) и расположена на 2–7 этаже в доме до 11 этажей.
*   **Время публикации:** пик объявлений приходится на будни, а также на период вокруг Нового года. Спад наблюдается в январе и мае.
*   **Срок продажи:** в среднем квартира продаётся за **92 дня**. Сделки короче 40 дней можно считать быстрыми, длиннее 200 дней – затяжными.
*   **Ценообразование:** главный фактор – общая площадь. Цена квадратного метра закономерно убывает с удалением от центра. Самые высокие цены – в Санкт‑Петербурге, самые низкие – в отдалённых посёлках.
*   **Динамика цен:** после спада в 2014–2017 гг. (связанного с кризисом) рынок перешёл к устойчивому росту.
*   **География предложения:** основная масса объектов сосредоточена в Санкт‑Петербурге и ближайших городах-спутниках (Кудрово, Мурино, Шушары).

## Запуск проекта

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/arseniybartenev/real_estate_eda.git
    ```
2.  Перейдите в папку проекта:
    ```bash
    cd real_estate_eda
    ```
3.  Установите необходимые зависимости:
    ```bash
    pip install -r requirements.txt
    ```
4.  Запустите Jupyter Notebook и откройте файл `notebooks/research_of_the_market_value_of_real_estate.ipynb`.
